1.  **Convert to String:** The most straightforward and robust way to count digits, especially for `0`, is to convert the number to a string and find its length. This handles `0` correctly (length of "0" is 1) and works for all other natural numbers.

    ```python

    n = read(int)
    # Convert the integer to a string
    n_str = str(n)
    # The number of digits is the length of the string
    num_digits = len(n_str)

    # Print the result in the specified format
    print(f"El nombre de digits de {n} es {num_digits}.")
    ```
