**Problem Summary:**
The problem "Nombre de dígits (1)" requires writing a program that reads a natural number `n` and outputs the total number of digits it contains. For example, if `n` is 123, the output should be 3; if `n` is 0, the output should be 1.

**Solution Approach:**
The provided Python program employs a standard arithmetic approach to count digits. It handles the special case of `n=0` by directly assigning 1 digit. For positive numbers, it iteratively divides the number by 10 (integer division) and increments a counter until the number becomes 0. This effectively counts how many times a digit can be "removed" until no digits are left.
