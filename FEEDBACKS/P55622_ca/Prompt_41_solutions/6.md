2.  **Alternative Numerical Approach (for learning/comparison):** While the current string-based solution is excellent, it might be beneficial to also consider and implement a solution that uses integer arithmetic. This would involve repeatedly dividing the number by 10 until it becomes 0, counting how many divisions were performed. This is demonstrated in many of the other provided example solutions.

    Here's an example of how a numerical approach might look using `easyinput`:

    ```python
    from easyinput import read

    def main():
        n = read(int)
        original_n = n # Store the original number for printing
        count = 0

        if n == 0:
            count = 1
        else:
            while n > 0:
                n //= 10  # Integer division
                count += 1

        print(f"El nombre de digits de {original_n} es {count}.")

    main()
    ```
