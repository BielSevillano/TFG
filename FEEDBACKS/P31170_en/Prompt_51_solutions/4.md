**3. Suggestions for Improvement:**

1.  **Strict Adherence to Output Format:** This is the most crucial change. Modify the `print` statement to match the expected format. Based on common practice and reference solutions, the format `n*i = result` is typical. Here are a few ways to achieve this:
    *   **Using f-strings (Python 3.6+):**
        ```python
        print(f"{x}*{i} = {x*i}")
        ```
    *   **Using string concatenation:**
        ```python
        print(str(x) + "*" + str(i) + " = " + str(x*i))
        ```

        ```python
        # If 'X*Y = Z' is required, a simple concatenation like the second option is best.
        ```

2.  **Consider `input()` for Simplicity:** While `sys.stdin.readline()` works, for simple single-line integer input like this, `n = int(input())` is often more idiomatic and concise in Python, as it automatically strips the newline character.
