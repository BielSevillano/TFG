#### 1. Brief Summary of the Problem and Solution Approach

The problem requires writing a program that reads a natural number `n` (between 1 and 9) and prints its multiplication table from 1 to 10. The provided solution approaches this by:
1.  Reading the input number `n` using `sys.stdin.readline()` and converting it to an integer.
2.  Using a `for` loop to iterate through numbers from 1 to 10 (inclusive).
3.  Inside the loop, calculating the product `n * i` for the current iteration `i`.
4.  Printing the multiplication expression and its result in the format `n*i = result` using the `print()` function with `sep=''` to control spacing.
