This program addresses the problem of determining and printing the number of digits in a given natural number.

**Solution Approach:**

The provided solution reads the input number as a string and then calculates its length using the `len()` function. This length directly corresponds to the number of digits in the input number. Finally, it prints a formatted string that includes the original number and its digit count.

*   **Simplicity and Readability:** The code is very straightforward and easy to understand. The use of `input()` and `len()` is intuitive for this task.

*   **Conciseness:** The solution is very short, achieving the desired outcome with minimal lines of code.
