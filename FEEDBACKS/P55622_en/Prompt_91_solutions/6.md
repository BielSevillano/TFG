1.  **Convert to String:** This is often the simplest and most readable method.

    *   Then, use `len()` on the string to get the number of digits.

    ```python
    from easyinput import read

    n = read(int)
    # Convert the integer to a string to find its length

    ```

2.  **Iterative Division (Loop):** This approach repeatedly divides the number by 10 until it becomes 0, counting how many divisions were performed.
    *   Handle the special case of `n = 0` (which has 1 digit).
    *   Use a `while` loop.

    ```python
    from easyinput import read

    n = read(int)
    original_n = n  # Store the original number for output
    digit_count = 0

    if n == 0:
        digit_count = 1
    else:
        # For positive numbers, repeatedly divide by 10
        while n > 0:
            n //= 10  # Integer division
            digit_count += 1

    print("The number of digits of", original_n, "is", digit_count, ".")
    ```

    from easyinput import read
