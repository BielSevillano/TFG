2.  **Mathematical Approach (for demonstration of numerical logic):**
    If the intention were to practice numerical manipulation rather than string manipulation, a common approach is to repeatedly divide the number by 10 until it becomes 0, counting the divisions. This approach handles the input as an integer.

    ```python
    n_str = input() # Read as string first to handle potential leading zeros if needed (though problem implies natural number)
    n = int(n_str)
    
    if n == 0:
        print("The number of digits of 0 is 1.")
    else:
        count = 0
        temp_n = n # Use a temporary variable to preserve original input for printing
        while temp_n > 0:
            temp_n //= 10
            count += 1
        print(f"The number of digits of {n} is {count}.")
    ```
    *Note: The problem statement implies a natural number, so `n=0` might not be a strictly required input according to some definitions of natural numbers, but it's good practice to consider.*

3.  **Using `math.log10` (for an even more mathematical approach):**
    For positive integers `n`, the number of digits is `floor(log10(n)) + 1`.

    ```python
    import math
    
    n_str = input()
    n = int(n_str)
    
    if n == 0:
        print("The number of digits of 0 is 1.")
    else:
        num_digits = int(math.floor(math.log10(n))) + 1
        print(f"The number of digits of {n} is {num_digits}.")
    ```
    *This approach is more complex for this simple problem but is a useful mathematical shortcut.*
