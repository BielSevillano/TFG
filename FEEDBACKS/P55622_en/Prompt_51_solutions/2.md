2.  **Alternative: String Conversion (for readability and conciseness):** A very common and often more readable approach in Python is to convert the number to a string and then get its length. This is often preferred for its simplicity.

    ```python
    from easyinput import read

    n = read(int)
    # Convert to string and get length
    num_digits = len(str(n))
    print("The number of digits of", n, "is", num_digits, ".")
    ```
    This approach is significantly shorter and leverages Python's built-in string functionalities.

3.  **Alternative: Using `math.log10` (for mathematical elegance):** For positive integers, the number of digits can also be calculated using logarithms.

    ```python
    from easyinput import read
    import math

    n = read(int)
    if n == 0: # Special case for 0, which has 1 digit
        num_digits = 1
    else:
        num_digits = math.floor(math.log10(n)) + 1
    print("The number of digits of", n, "is", num_digits, ".")
    ```
    This is a more mathematical approach but might be less intuitive for beginners and requires handling the edge case of 0 separately.
