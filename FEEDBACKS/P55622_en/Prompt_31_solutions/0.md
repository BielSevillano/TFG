3.  **Exploring Numerical Approaches (for learning):** While the string conversion is elegant, for educational purposes or if string conversion were disallowed, a numerical approach could be implemented. This would involve repeatedly dividing the number by 10 and incrementing a counter until the number becomes zero. For example:

    ```python
    from easyinput import read

    entrada = read(int)
    count = 0
    if entrada == 0:
        count = 1
    else:
        temp = abs(entrada) # Handle potential negatives if applicable

            temp //= 10
            count += 1
    print(f"The number of digits of {entrada} is {count}.")
    ```
