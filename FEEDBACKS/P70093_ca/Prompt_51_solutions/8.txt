#### 1. Problem and Solution Approach Summary

The problem "Fusió de taules" asks to implement a function that merges two pre-sorted lists of numbers (`v1` and `v2`) into a single, combined sorted list. The merged list should preserve all duplicate elements from the original lists. For example, merging `[2, 3, 5]` and `[3, 5, 5, 7]` should produce `[2, 3, 3, 5, 5, 5, 7]`.

The submitted `fusio` function employs the standard **two-pointer (or merge) algorithm** to solve this problem. It iterates through both input lists simultaneously, comparing elements at the current pointers and appending the smaller one to a new result list. This process continues until one of the lists is exhausted, after which any remaining elements from the other list are simply appended to the result.
