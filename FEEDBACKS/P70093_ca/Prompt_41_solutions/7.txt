**Summary of the Problem and Solution Approach:**

The problem "Fusió de taules" asks for a function that merges two already sorted lists (or vectors/arrays) of numbers into a single new sorted list. It's crucial that duplicate elements are retained. For example, merging `[2, 3, 5]` and `[3, 5, 5, 7]` should result in `[2, 3, 3, 5, 5, 5, 7]`.

The student's `fusio` function implements a classic two-pointer merge algorithm. It iterates through both input lists simultaneously using two index pointers (`i1` and `i2`). In each step, it compares the elements at the current pointers and appends the smaller one to a new `llista_definitiva`. The corresponding pointer is then incremented. Once one list is fully processed, any remaining elements from the other list are simply appended to the result.
