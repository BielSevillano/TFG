### Feedback on "Fusió de taules" Program

**Problem Summary:**
The problem "Fusió de taules" requires implementing a function `fusio` that merges two pre-sorted lists (`v1` and `v2`) of floating-point numbers into a single sorted list. The merged list should retain all duplicate elements. For example, merging `[2, 3, 5]` and `[3, 5, 5, 7]` should yield `[2, 3, 3, 5, 5, 5, 7]`. The input lists are guaranteed to be sorted in ascending order.

**Solution Approach:**
The `fusio` function uses a classic two-pointer merge algorithm. It initializes two pointers (`i1`, `i2`) to the beginning of `v1` and `v2` respectively, and an empty list `llista_definitiva` to store the result. It iteratively compares the elements pointed to by `i1` and `i2`, appending the smaller element to `llista_definitiva` and advancing the corresponding pointer. If elements are equal, it appends the element from `v2` (or `v1`, the choice doesn't matter for correctness) and advances `i2`. Once one list is exhausted, it efficiently appends the remaining elements of the other list to `llista_definitiva`. This is a standard and optimal approach for this problem.

The program also includes a `main` function with helper functions (`llegir_dades`) to handle input reading and output printing, which is typical for local testing in competitive programming.
