**Suggestions for Improvement:**

1.  **Explicitly stop at the period:** While not strictly necessary for correctness in this problem, a more robust approach that adheres precisely to the "ended in a period" condition would be to stop counting once the period is encountered. This can be achieved by using a `while` loop and checking for the period within the loop condition or body.

    Here's an example of how this could be done:

    ```python
    s = input()
    c = 0
    for char in s:
        if char == 'a':
            c += 1
        if char == '.': # Stop processing after encountering the period
            break
    print(c)
    ```

    Or using a `while` loop:

    ```python
    s = input()
    c = 0
    i = 0
    while i < len(s) and s[i] != '.':
        if s[i] == 'a':
            c += 1
        i += 1
    print(c)
    ```
