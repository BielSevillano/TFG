### 3. Suggestions for Improvement

1.  **Encapsulate Logic in a Function:** To improve code organization and follow best practices, the core logic could be placed within a function.

    ```python
    from easyinput import read

    def print_multiplication_table(num):
        """Prints the multiplication table for a given number."""
        for i in range(1, 11):
            print(num, "*", i, " = ", num * i, sep='')

    if __name__ == "__main__": # Standard entry point guard
        n = read(int)
        print_multiplication_table(n)
    ```

2.  **Consider Using f-strings for Output (Python 3.6+):** While `sep=''` works perfectly, f-strings often offer a more readable and flexible way to format strings, especially when mixing variables and literal text.

    ```python
    from easyinput import read

    n = read(int)
    for i in range(1, 11):
        print(f"{n}*{i} = {n*i}")
    ```
    This achieves the same compact output as `sep=''` but with a different syntax, which some find more expressive.
