    For better organization and reusability, especially as programs grow, it's good practice to wrap core logic in functions.
    ```python
    def print_multiplication_table(num):
        for i in range(1, 11):
            print(num, "*", i, " = ", num*i, sep="")

    n = int(input())
    print_multiplication_table(n)
    ```

4.  **Use f-strings for output formatting (modern Pythonic way):**
    F-strings (formatted string literals) provide a very clean and readable way to embed expressions inside string literals for printing. This is often preferred over using multiple arguments with `sep=""` or string concatenation.
    ```python
    # n = int(input()) # (assuming n is already read)
    for i in range(1, 11):
        print(f"{n}*{i} = {n*i}") # Using an f-string for formatting
    ```
