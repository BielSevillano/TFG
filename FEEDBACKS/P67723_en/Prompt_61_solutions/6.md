## Feedback on GCD Program

This program correctly calculates the greatest common divisor (GCD) of two strictly positive natural numbers. The problem statement is clear: find the largest number that divides both input numbers without leaving a remainder.

### Solution Approach:

The provided solution implements the **Euclidean Algorithm**. This is a highly efficient method for computing the GCD. The core idea is based on the principle that the GCD of two numbers does not change if the larger number is replaced by its difference with the smaller number. Repeatedly applying this process, or more efficiently, using the modulo operator, eventually leads to the GCD.

The code snippet `while b != 0: r=a%b; a=b; b=r` is a concise and standard implementation of the Euclidean Algorithm using the modulo operator.
