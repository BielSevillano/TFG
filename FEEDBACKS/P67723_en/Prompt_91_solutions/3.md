The problem asks to write a program that calculates the greatest common divisor (GCD) of two strictly positive natural numbers.

The provided solution implements the **Euclidean algorithm** to find the GCD. This algorithm is efficient and widely used for this purpose. It works by repeatedly applying the modulo operation. The core idea is that the GCD of two numbers `a` and `b` (where `a > b`) is the same as the GCD of `b` and the remainder of `a` divided by `b` (`a % b`). This process continues until the remainder is 0, at which point the GCD is the last non-zero remainder.

*   **Correctness:** The code correctly implements the Euclidean algorithm, which is a proven method for finding the GCD.

    ```python
    x = read(int)
    y = read(int)

    if x <= 0 or y <= 0:
        print("Error: Input numbers must be strictly positive.")
    else:
        a = x
        b = y
        while b != 0:
            r = a % b
            a = b
            b = r

        print()
    ```
