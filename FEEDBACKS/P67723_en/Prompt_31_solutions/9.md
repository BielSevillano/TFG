This program calculates the greatest common divisor (GCD) of two strictly positive natural numbers.

**Solution Approach:**

The program implements the Euclidean algorithm to find the GCD. The core idea of the Euclidean algorithm is that the GCD of two numbers does not change if the larger number is replaced by its difference with the smaller number. This process is repeated until one of the numbers becomes zero, at which point the other number is the GCD. The provided code uses the modulo operator (`%`) which is a more efficient way to achieve the same result as repeated subtraction.
